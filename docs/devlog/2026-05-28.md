# 2026-05-28 开发日志

## Media 表扩展 EXIF 与编码元数据

**背景**：之前 `Media` 表只落库 `width / height / duration_ms`，而 `MediaInfoUtils.get_media_info` 实际还提取了 `bitrate / fps / codec / has_audio / taken_at / gps`,这些都被丢弃。同时反映出一个现象:很多视频在浏览器里"只有声音没画面",疑似 HEVC/AV1 等浏览器不支持的编码 —— 但 DB 里没有 `video_codec` 字段,无从筛查。

**改动**：

1. `backend/app/models/__init__.py` 的 `Media` 新增 12 列(全 nullable):
   - 基础: `taken_at(DateTime, indexed)`、`gps_lat / gps_lng(Float)`、`orientation(Int)`
   - 相机: `camera_make / camera_model(String 64)`、`lens(String 128)`
   - 编码: `video_codec / audio_codec(String 32)`、`has_audio(Int 0|1)`、`fps(Float)`、`bitrate(Int)`
2. `backend/app/utils/__init__.py`:
   - `get_media_info` 返回 dict 的键改为与 Model 字段一一对齐(扁平 `gps_lat/gps_lng` 替代嵌套 `gps`,`video_codec` 替代 `codec`),`taken_at` 返回 `datetime` 而非字符串。
   - 视频分支补 `audio_codec`,并从 `format.tags` 读 `make / model / com.apple.quicktime.make/model` 填相机字段。
   - 新增 `_extract_exif_camera`,从 PIL EXIF 读 Make/Model/Orientation 和 ExifIFD 里的 LensModel。
3. `backend/app/services/media_service.py` 的 `process_standalone_file` 与 `replace_media_file` 把 12 个新字段一并写入 Media。
4. `backend/alembic/versions/20260528_1000_d0e1f2a3b4c5_media_exif_fields.py`:`batch_alter_table` 加列 + `taken_at` 索引。
5. `backend/app/schemas/media.py` 的 `MediaResponse` 暴露新字段给前端。
6. `backend/scripts/backfill_media_exif.py`:对存量 Media 重跑 `get_media_info`,默认只填空字段,`--force` 覆盖,`--limit N` 限制范围。跳过预览帧。

**下一步**:跑完回填后用 `SELECT video_codec, COUNT(*) FROM media WHERE video_codec IS NOT NULL GROUP BY video_codec` 看一下 HEVC/AV1 分布,再决定是预转码 H.264 副本还是请求时实时转码。

## 暗色主题重做（中性石墨 + 系统字体）

**背景**：原暗色主题混用红棕背景(`rgb(32,26,27)`)、中性灰(`rgb(32,32,32)`)、紫红渐变和 indigo primary，色相冲突；卡片与背景几乎同色，靠描边区分层级，长时间看疲劳。字体走 Google Fonts 的 Noto Sans/Serif SC，小字号下笔画偏细看不清。

**改动**：

1. `vue/src/style.css`：`.dark` token 全面重做。
   - 三档背景统一中性灰：`--bg-primary #212121` / `--bg-secondary #2a2a2a` / `--bg-card #333333`，靠明度差分层而非描边。
   - 文本对比度提档：`--text-primary #f0f6fc` / `--text-secondary #c9d1d9` / `--text-muted #8b96a3`。
   - Primary 从 `#818cf8` 改为 `#7c93ff`(略带蓝紫，在中性底上不刺)。
   - 滚动条、`@theme inline` 的 select 暗色变量、`.tag-chip`、Milkdown Crepe 变量一并对齐新色板。
   - 字体回归系统默认：`system-ui, -apple-system, 'Segoe UI', 'PingFang SC', 'Microsoft YaHei', sans-serif`，display 与 body 统一(不再用 Serif 标题)。
2. `vue/index.html`：移除 Google Fonts 的 preconnect 和 `<link>`。
3. `vue/src/App.vue`：去掉紫红渐变背景，dark 态用纯 `var(--bg-primary)`。
4. Dashboard 卡片：`HeatmapCard / IssueBoard / TodoBoard / RecentMedia` 的 `dark:bg-black/20` → `dark:bg-[var(--bg-card)]`(#333)；`IssueColumn / TodoColumn` → `dark:bg-[var(--bg-secondary)]`(#2a2a2a)；`IssueCard / TodoCard` 的 `dark:bg-gray-800` → `dark:bg-[#3d3d3d]`(浮在面板之上)；HeatmapCard 的 Stat 小卡同步改为 `#3d3d3d`。
