   - `syncFromRemote()` 与增量分支的 MEDIA UPSERT：用 `rm.created_at?.let(::parseIsoToMs)` / `rm.updated_at?.let(::parseIsoToMs)` 替换无脑 `now()` 兜底，避免每次拉取都把 LWW 时间戳刷成同步当时的本地时间。

4. **`docs/sync-apply-prompt.md`**：末尾新增「已知差距」一节，列出 `since_id`、SSE、视频章节、消息合并/拆分等下次再做。
