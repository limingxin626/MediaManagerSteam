# 2026-04-14 开发日志

## Android 底部导航栏改为悬浮样式

### 背景

原先使用 Material3 标准 `NavigationBar` + `NavigationBarItem`，固定在屏幕底部，占满整个宽度。改为现代悬浮药丸样式导航栏，提升视觉品质。

### 实现内容

**`MainActivity.kt` — 替换底部导航栏实现**

- 移除 Scaffold 的 `bottomBar` 参数，改用 `Box` 布局叠加内容和悬浮导航栏
- 内容区域 (`AppNavHost`) 填满整个屏幕，导航栏浮在内容上方
- 使用 `AnimatedVisibility` + `slideInVertically`/`slideOutVertically` 实现导航栏进出动画
- 导航栏距屏幕边缘 24dp 水平内边距 + 12dp 底部间距 + 系统导航栏安全区

**新增 `FloatingNavBar` 组件**

- `Row` 容器 + `RoundedCornerShape(28.dp)` 圆角胶囊形状
- `shadow(8.dp)` 阴影 + 半透明 `surfaceContainer` 背景 (95% 不透明度)
- 选中项：`primary` 色 15% 透明度圆角背景高亮 + `primary` 色图标/文字
- 未选中项：透明背景 + `onSurfaceVariant` 色图标/文字
- 移除波纹效果 (`indication = null`)，保持简洁
- 图标 22dp，标签使用 `labelSmall` 字体
