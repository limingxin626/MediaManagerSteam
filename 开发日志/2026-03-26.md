# 2026-03-26 开发日志

## Message 页面 UI 优化

### 1. 日期分隔符
- 消息流中不同日期的消息之间新增日期分隔线
- 显示格式：今天/昨天/X月X日/YYYY年X月X日
- 居中显示，两侧有分割线

### 2. 消息流布局重构（flex 布局替代 fixed 定位）
- 将页面从 page-level scroll + fixed header/footer 改为 flex 列布局
- 顶部搜索栏和底部输入栏是 `shrink-0` 静态元素，不再 fixed
- 中间消息区域 `flex-1 overflow-y-auto` 是唯一可滚动区域
- 内容物理上无法超出到搜索栏和输入栏后面，无需半透明背景 hack
- 所有 `window.scrollTo` / `document.body.scrollHeight` 改为操作 scrollContainer ref
- IntersectionObserver 的 root 改为 scrollContainer
- 合并栏和"回到最新"按钮改为 sticky 定位

### 3. 媒体菜单按钮可见性
- 媒体项右上角菜单按钮改为跟随卡片 `group-hover` 显示
- 之前只有悬停按钮自身才显示，按钮太小难以发现

### 4. 3 图网格布局优化
- 从「1 大图在上 + 2 小图在下」改为「左 1 大图 + 右 2 小图」
- 使用 `grid-cols-3 grid-rows-2`，第一张图 `col-span-2 row-span-2`

### 5. MediaPreview 缩略图 URL 修复
- 缩略图条统一使用 `resolveUrl(item.thumb_url)`
- 之前图片类型用 `item.thumb_url || item.file_path`，未经过 resolveUrl 处理

### 6. Loading Skeleton
- 初始加载时用骨架屏占位（3 个卡片骨架），代替简单的旋转圆圈
- 加载更多时仍使用小型 spinner
