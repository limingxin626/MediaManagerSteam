# 2026-04-16 开发日志

## Android Mosaic 多图布局优化

### 背景

消息流中多图布局使用固定的计数→布局映射（`getGridLayout`），所有图片在行内分配相同宽度，不考虑实际宽高比，导致横图和竖图混排时裁切严重。改为 Telegram 风格 Mosaic 布局算法，根据图片宽高比动态计算最优行分配。

### 实现内容

**新增 `MosaicLayout.kt` — Mosaic 布局算法**

- 数据结构：`MosaicLayout`、`MosaicRow`、`MosaicItem`，支持 ROWS（纯行）和 LEFT_COLUMN（左大图+右侧堆叠）两种布局类型
- 宽高比分类：wide（>1.2）、narrow（<0.8）、square（0.8-1.2）
- 2 图启发式：两张相似宽图上下排列，否则按比例左右并排
- 3 图启发式：首图宽→上方全宽+下方两列；首图窄/方→L 型布局
- 4+ 图优化搜索：遍历所有行分配方案（最多 4 行，每行最多 4 个），硬约束过滤（行高 ≥ 容器宽度 28%、单图面积 ≥ 容器面积 4%），在满足约束的方案中选面积方差最小的

**修改 `MessageCard.kt` — 渲染集成与卡片样式**

- 替换旧的 `GridLayoutSpec`、`getGridLayout()`、`RowBasedGrid()` 为 `MosaicRowsGrid` 和 `MosaicLeftColumnGrid` composable
- 卡片最大高度从 500dp 提升到 800dp，适应多图场景
- 卡片宽度从 100% 改为 90% 居中显示，类似微信聊天风格
- 保留 "+N" 溢出显示、视频标识、星标等现有功能

### 涉及文件

- `android/.../ui/components/MosaicLayout.kt`：新增，Mosaic 布局算法
- `android/.../ui/components/MessageCard.kt`：删除旧 GridLayout，新增 Mosaic 渲染器，卡片 90% 居中

## Vue 前端 Mosaic 多图布局

### 背景

Vue 前端 MessageCard 使用固定 CSS Grid 布局（grid-cols-2/3），所有图片统一 aspect-square，不考虑实际宽高比。移植 Android 端的 Telegram 风格 Mosaic 算法到前端，实现与 Android 一致的动态布局。

### 实现内容

**后端 schema 修改**

- `MessageMediaItem` 新增 `width`、`height` 字段（从 Media 模型透传），让前端获取宽高比信息

**新增 `vue/src/utils/mosaic.ts` — Mosaic 布局算法（TypeScript）**

- 从 Android `MosaicLayout.kt` 直接移植，算法逻辑完全一致
- 导出 `calculateMosaicLayout(ratios, containerWidth)` 函数
- 支持 ROWS 和 LEFT_COLUMN 两种布局类型
- 1 图：保持原始宽高比显示
- 2 图：相似宽图上下排列，否则按比例左右并排
- 3 图：首图宽→上全宽+下两列；首图窄/方→L 型布局
- 4+ 图：遍历行分配方案，面积方差最小化

**修改 `vue/src/components/MessageCard.vue` — 渲染集成**

- 删除旧的 `mediaGridClass`/`mediaItemClass` CSS Grid 布局
- 单图使用 CSS `aspect-ratio` 保持原始比例
- 多图使用 `flex` 布局 + 内联 `style` 实现 Mosaic 行和列权重
- ROWS 类型：flex 列容器套 flex 行，每行高度由算法决定
- LEFT_COLUMN 类型：flex 行容器，左大图 + 右侧 flex 列堆叠
- 保留视频标识、星标、时长、"+N" 溢出、菜单等所有现有功能

**前端类型修改**

- `MessageMediaItem` 新增 `width: number | null`、`height: number | null`

### 涉及文件

- `backend/app/schemas/message.py`：MessageMediaItem 新增 width/height
- `vue/src/utils/mosaic.ts`：新增，Mosaic 布局算法
- `vue/src/components/MessageCard.vue`：替换 Grid 为 Mosaic 布局
- `vue/src/types.ts`：MessageMediaItem 新增 width/height
